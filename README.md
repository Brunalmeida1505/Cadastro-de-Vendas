# Cadastro de Vendas
Algo simples, mas que para um pequeno varejista pode agilizar muito o preenchimento de uma planilha de vendas.
O arquivo utiliza uma ação macro do Excel, automatizando o cadastro de produtos.

## COMO USAR
Basta preencher nas cédulas A2, B2 e C2 os itens e clicar no botão logo ao lado, escrito "Cadastrar"
Ao clicar ele é automaticamente inserido abaixo, criando uma planilha de produtos.
